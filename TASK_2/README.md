Unemployment Analysis with Python

Project Overview

This project analyzes unemployment data using Python and identifies
unemployment trends and Covid-19 impact.

Objectives

-   Data Cleaning
-   Exploratory Data Analysis (EDA)
-   Data Visualization
-   Trend Analysis
-   Covid-19 Impact Analysis

Technologies Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn
-   Jupyter Notebook

Project Files

Project Folder/ ├── Unemployment in India.csv ├──
Unemployment_Rate_upto_11_2020.csv ├── cleaned_main_dataset.csv ├──
cleaned_covid_dataset.csv ├── Unemployment_Analysis_Project.ipynb └──
README.txt

Workflow

1.  Import Libraries
2.  Load Dataset
3.  Dataset Overview
4.  Data Cleaning
5.  EDA
6.  Visualization
7.  Covid Impact Analysis
8.  Insights
9.  Conclusion

Key Insights

-   Unemployment increased during Covid period.
-   Different regions experienced different unemployment levels.
-   Trend analysis showed variations over time.
-   Visualizations improved understanding.
-   Clean data produced better analysis.

Conclusion

The project demonstrates practical data analysis using Python to
understand unemployment patterns.
